# Proyecto ETL Anime/Manga — Minería de Datos

## Integrantes
- Grupo 10 — Chavarro & Pascuas

## Descripción
Pipeline de minería de datos que extrae, transforma y carga
información de anime, manga y personajes desde la API de MyAnimeList
(Jikan API) hacia PostgreSQL en la nube, con dashboards interactivos
y modelos de Machine Learning.
